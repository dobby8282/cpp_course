## 🔧 컴파일 및 실행

### 개별 파일 컴파일
```bash
g++ -std=c++17 -o output_name filename.cpp
./output_name
```

### 모든 예제 컴파일 (Makefile 사용)
```bash
# 현재 챕터의 모든 예제 컴파일
make chapter01

# 특정 예제만 컴파일
make chapter01/example_name
```
